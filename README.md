# Ukraine Critical Raw Materials Occurrence Catalogue

This repository contains the harmonised catalogue of Ukrainian Critical Raw Material (CRM) occurrences used in the study:

Krijthe et al. – Systematic Literature Review of Ukraine’s Critical Raw Material Occurrences.

The dataset contains 277 documented CRM occurrences compiled from peer-reviewed literature, Ukrainian geological documentation, licensing records, and institutional reports.

Fields included:
- CRM category
- occurrence name
- location (oblast)
- deposit type
- geological evidence score
- data quality score
- classification alignment score
- screening results
